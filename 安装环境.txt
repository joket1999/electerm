windows 打包环境

到 "下载" 文件夹

1.安装里面的 NDP462-KB3151802-Web.exe 、visualcppbuildtools_full.exe、python-2.7.15.amd64.exe、Git-2.19.2-64-bit.exe

2.设置python环境变量 在PATH加上python安装路径，如：C:\phthon

3.打开Git Bash 

4.创建一个文件夹：mkdir project && cd project

5.克隆urlssh项目到本地：git clone ssh://git@html.dev.dockeradmin.com:10022/joshua/sshurl.git

6.进入项目：cd sshurl

7.切换到release分支：git checkout release

8.设置npm打包环境：npm config set python python2.7、npm config set msvs_version 2015

9.安装node依赖：npm install

10.打包 npm run release
